This is Analysis Soft for Dual Compton Cameras Experiment @ OIST.
written by Hiroki Yoneda -- 2017 04 05

====== install ======
- install share library
1. cd share
2. mkdir build
3. cmake .. (if it does not work, add "--DROOT_AUTOSEARCH")

- install imaging soft
1. cd ImagingSoft
2. mkdir build
3. cmake .. (if it does not work, add "--DROOT_AUTOSEARCH")
